# Jupuk AIPI (Get IP Address)

This repository is the source code of https://jupuk-aipi.herokuapp.com, a website that will help people gather IP address information when ISP blocked it. 

# Getting Started

The only guide that will be available in here is how to run this repository.

## Requirements

This repository only requires one thing: [node.js](https://nodejs.org)

## Installation

1. Open the root directory of this repository
2. Open terminal and run:

  ```sh
  npm install
  ```

  Or

  ```sh
  yarn
  ```

## Run Development Server

This repository uses `concurrently` to run multiple process.

```sh
yarn develop
```

## Deployment

To deploy the apps, the only script needed is:

```sh
yarn start
```
